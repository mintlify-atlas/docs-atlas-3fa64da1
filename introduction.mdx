## What is GreenGuard?

GreenGuard is a comprehensive urban incident reporting platform that connects citizens, municipalities, and local businesses to improve community safety and infrastructure. Citizens report issues like potholes, fallen poles, and broken sidewalks, earn points through gamification, and redeem rewards from partner stores. Municipalities track and resolve incidents efficiently, while stores gain visibility and customer engagement.

## Technology Stack

GreenGuard is built with modern, production-ready technologies:

- **Backend**: Spring Boot 3.5.8 REST API with Java 21
- **Frontend**: Angular 19.2 with standalone components
- **Database**: PostgreSQL with JPA/Hibernate
- **Notifications**: Firebase Cloud Messaging (FCM)
- **AI Integration**: Incident classification and analysis
