---
title: 'List Incident Reports'
description: 'Retrieve a list of incident reports with optional filtering'
api: 'GET /reporte'
---

## Overview

Retrieve a paginated list of incident reports. You can filter by status, district, incident type, date range, and user.

## Query Parameters

<ParamField query="page" type="integer" default="0">
  Page number for pagination (0-indexed)
</ParamField>

<ParamField query="size" type="integer" default="20">
  Number of reports per page
</ParamField>

<ParamField query="estado" type="string">
  Filter by report status:
  - `PE` - Pendiente (Pending)
  - `EP` - En Proceso (In Process)
  - `RE` - Resuelto (Resolved)
  - `CA` - Cancelado (Cancelled)
</ParamField>

<ParamField query="idDistrito" type="integer">
  Filter by district ID
</ParamField>

<ParamField query="idTipoInci" type="integer">
  Filter by incident type ID
</ParamField>

<ParamField query="idTipoClasi" type="integer">
  Filter by classification type ID
</ParamField>

<ParamField query="idUsu" type="integer">
  Filter by user ID (reports created by specific user)
</ParamField>

<ParamField query="fechaInicio" type="date">
  Filter reports from this date (format: YYYY-MM-DD)
</ParamField>

<ParamField query="fechaFin" type="date">
  Filter reports until this date (format: YYYY-MM-DD)
</ParamField>

<ParamField query="sort" type="string" default="repoRegistado,desc">
  Sort field and direction (e.g., "repoRegistado,desc" or "estado,asc")
</ParamField>

## Example Request

```bash
# Get all reports
curl -X GET "https://api.greenguard.com/reporte" \
  -H "Authorization: Bearer YOUR_TOKEN"

# Get pending reports in a specific district
curl -X GET "https://api.greenguard.com/reporte?estado=PE&idDistrito=5&page=0&size=10" \
  -H "Authorization: Bearer YOUR_TOKEN"

# Get reports by user with date range
curl -X GET "https://api.greenguard.com/reporte?idUsu=1&fechaInicio=2024-01-01&fechaFin=2024-03-31" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
