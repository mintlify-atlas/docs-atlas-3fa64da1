## Overview

Get a list of all categories (categorías) available in the GreenGuard platform. Categories are used to organize coupons and rewards that users can earn through environmental actions.

<Info>
  Categories help users browse and filter available coupons by type of business or reward.
</Info>

## Request

This endpoint requires no parameters.

```bash
GET /categoria
```

## Response

<ResponseField name="categories" type="array">
  Array of category objects
  <ResponseField name="idCate" type="integer">
    Unique category identifier
  </ResponseField>
  <ResponseField name="descCate" type="string">
    Category description/name
  </ResponseField>
</ResponseField>

## Data Model

The category data structure corresponds to the `tb_categoria` table:

| Field | Type | Description |
|-------|------|-------------|
| `id_cate` | INTEGER | Primary key, auto-increment |
| `desc_cate` | VARCHAR | Category name/description |

## Usage in Coupons

When creating or updating a coupon, reference a category using its `idCate`:

```json
{
  "nombreCupon": "15% descuento en productos eco",
```

## Common Categories

The platform typically includes these coupon categories:

- **Restaurantes**: Dining and food establishments offering eco-friendly options
- **Retail**: Stores selling sustainable products or eco-friendly merchandise
- **Transporte**: Transportation services (bike rentals, electric vehicle charging, public transit)
- **Entretenimiento**: Entertainment venues with environmental focus
- **Servicios**: Services like eco-cleaning, sustainable consulting, green renovations
- **Supermercados**: Grocery stores with organic/local product sections
- **Tecnología**: Electronics, gadgets with energy efficiency or recycled materials
- **Salud y Bienestar**: Health and wellness centers with natural/organic focus

## Category-Based Filtering

Allow users to browse coupons by category to find rewards that interest them:

```javascript
// Get all categories and let user select
const categories = await fetch('/categoria').then(r => r.json());

// Display categories in UI
categories.forEach(cat => {
  console.log(`${cat.idCate}: ${cat.descCate}`);
});

// Fetch coupons for selected category
const selectedCategoryId = 1; // Restaurantes
const coupons = await fetch(`/cupon/categoria/${selectedCategoryId}`)
  .then(r => r.json());
```

<Tip>
  Categories help users discover relevant rewards. Consider showing category icons in your UI to improve the browsing experience.
</Tip>

## Point System Integration

Users earn points by submitting environmental reports. These points can be redeemed for coupons organized by category:

1. User submits environmental report
2. System awards points based on report validation
3. User browses available coupons by category
4. User redeems points for desired coupon
5. Coupon can be used at participating stores

<Note>
  The number of points required varies by coupon and is set by the participating business or platform administrators.
</Note>
