## Overview

Get a list of all classification types (tipos de clasificación) available in the GreenGuard platform. Classification types are used to further categorize incidents or materials in environmental reports.

<Info>
  Classification types provide additional context for incidents and are typically used alongside incident types.
</Info>

## Request

This endpoint requires no parameters.

```bash
GET /clasificacion
```

## Response

<ResponseField name="classificationTypes" type="array">
  Array of classification type objects
  <ResponseField name="idTipoClasi" type="integer">
    Unique classification type identifier
  </ResponseField>
  <ResponseField name="descTipoClasi" type="string">
    Classification type description
  </ResponseField>
</ResponseField>

When creating a report, you can include classification type to provide additional context:

```json
{
  "titulo": "Botellas plásticas acumuladas",
  "descripcion": "Gran cantidad de botellas PET en el parque",
  "tipoIncidente": {
    "idTipoInci": 1
  },

    "idTipoClasi": 3
  },
  "distrito": {
    "idDistrito": 2
  },
  "direccion": "Parque Kennedy"

```

## Data Model

The classification type data structure corresponds to the `tb_tipo_clasificacion` table:

| Field | Type | Description |
|-------|------|-------------|
| `id_tipo_clasi` | INTEGER | Primary key, auto-increment |
| `desc_tipo_clasi` | VARCHAR | Classification type description |

## Classification Categories

The platform typically includes these classification types:

### Waste Classifications

- **Orgánico**: Organic waste (food scraps, garden waste, biodegradable materials)
- **Inorgánico**: Inorganic waste (general non-recyclable trash)
- **Reciclable**: Recyclable materials (plastic, glass, paper, cardboard, metal)

### Special Handling

- **Peligroso**: Hazardous waste (chemicals, batteries, toxic materials)
- **Electrónico**: Electronic waste (computers, phones, appliances)
- **Construcción**: Construction debris (concrete, wood, metal scraps)
- **Médico**: Medical waste (requires special disposal procedures)

## Combined Usage

Classification types work together with incident types to provide comprehensive categorization:

```javascript
// Example: Reporting recyclable waste accumulation
{
  "tipoIncidente": {
    "idTipoInci": 1,           // Acumulación de basura
    "descTipoInci": "Acumulación de basura"
  },
  "tipoClasificacion": {
    "idTipoClasi": 3,          // Reciclable
    "descTipoClasi": "Reciclable"
  }
}

// Example: Hazardous waste dumping
{
  "tipoIncidente": {
    "idTipoInci": 5,           // Vertido ilegal
    "descTipoInci": "Vertido ilegal"
  },
  "tipoClasificacion": {
    "idTipoClasi": 4,          // Peligroso
    "descTipoClasi": "Peligroso"
  }
}
```

<Tip>
  Use classification types to help route reports to the appropriate authorities. Hazardous waste requires different handling than recyclable materials.
</Tip>

<Warning>
  Reports classified as "Peligroso" (hazardous) should be prioritized and may require immediate action from specialized authorities.
</Warning>
