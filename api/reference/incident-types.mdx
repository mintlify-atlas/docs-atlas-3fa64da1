## Overview

Get a list of all incident types (tipos de incidentes) available in the GreenGuard platform. Incident types categorize environmental reports by the nature of the issue being reported.

<Info>
  Incident types are reference data used when creating environmental reports. This data is typically static.
</Info>

## Request

This endpoint requires no parameters.

```bash
GET /incidente
```

## Response

<ResponseField name="incidentTypes" type="array">
  Array of incident type objects
  <ResponseField name="idTipoInci" type="integer">
    Unique incident type identifier
  </ResponseField>
  <ResponseField name="descTipoInci" type="string">
    Incident type description
  </ResponseField>
</ResponseField>

## Usage in Reports

When creating an environmental report, reference an incident type using its `idTipoInci`:

```json
{
  "titulo": "Basura en parque público",
  "descripcion": "Gran acumulación de basura cerca del área de juegos",
  "tipoIncidente": {
    "idTipoInci": 1
  },
  "distrito": {
    "idDistrito": 2
  },
  "direccion": "Parque Kennedy"
}
```

## Data Model

The incident type data structure corresponds to the `tb_tipos_incidentes` table:

| Field | Type | Description |
|-------|------|-------------|
| `id_tipo_inci` | INTEGER | Primary key, auto-increment |
| `desc_tipo_inci` | VARCHAR | Incident type description |

## Common Incident Types

The platform typically includes these incident categories:

- **Acumulación de basura**: Waste accumulation in public areas
- **Contaminación del agua**: Water pollution in rivers, beaches, or water bodies
- **Contaminación del aire**: Air quality issues, smoke, or emissions
- **Deforestación ilegal**: Illegal tree cutting or forest clearing
- **Vertido ilegal**: Illegal dumping of waste or hazardous materials
- **Ruido excesivo**: Noise pollution from industrial or commercial sources
- **Daño a flora/fauna**: Damage to plants, trees, or wildlife
- **Otro**: Other environmental issues not covered by specific categories

<Tip>
  Present incident types in a dropdown or selection list when users create reports. This ensures consistent categorization and improves report processing.
</Tip>
